# Day 12 - Campus Recruitment Training (CRT)

## Topic

SQL Operators, Filtering, Sorting, and Group Functions

## Description

Today's practice focused on SQL query operations used for
filtering records, pattern matching, sorting data, handling
NULL values, and performing aggregate calculations.

The session also included working with date ranges, string
comparisons, subqueries, GROUP BY, HAVING, and ORDER BY
clauses using Employee and Department tables.

---

## Concepts Covered

### 1. BETWEEN Operator

* Numeric and String Range Search
* Date Range Filtering
* NOT BETWEEN

### 2. IN and NOT IN

* Selecting Multiple Values
* Excluding Multiple Values
* Subquery with IN

### 3. LIKE Operator

* Starts With (`'A%'`)
* Contains Character (`'%a%'`)
* Single Character Wildcard (`'_a%'`)

### 4. NULL Handling

* IS NULL
* IS NOT NULL

### 5. Sorting Data

* ORDER BY Ascending
* ORDER BY Descending
* Multiple Column Sorting

### 6. Aggregate Functions

* COUNT()
* GROUP BY
* HAVING Clause

### 7. Joins and Grouping

* LEFT JOIN
* Counting Related Records
* Grouping Joined Data

---

## Operations Performed

* Retrieved employees using BETWEEN and NOT BETWEEN
* Filtered employees based on joining dates
* Used IN and NOT IN for city-based searches
* Applied LIKE patterns for employee names
* Checked NULL and NOT NULL values
* Sorted employee records by name and salary
* Counted employees city-wise using GROUP BY
* Used HAVING clause for filtered grouped results
* Attempted JOIN operations between Employee and Department tables
* Practiced subqueries using IN operator
