# Day 13 - Campus Recruitment Training (CRT)

## Part 1

### Topic

SQL Constraints and Joins

## Description

Today's practice focused on advanced SQL concepts including
Primary Keys, Composite Keys, Foreign Keys, Table Alterations,
and various Join operations.

The session also included creating relational databases,
inserting records, modifying table structures, and retrieving
data using different SQL joins.
