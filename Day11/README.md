# Day 11 - SQL Transactions and Aggregate Functions

## Overview

On Day 11 of Campus Recruitment Training (CRT), I practiced SQL transaction control statements, data retrieval queries, aggregate functions, subqueries, and various built-in mathematical functions.

This session helped strengthen understanding of database operations and SQL query writing.

---

## Topics Covered

### Transaction Control Language (TCL)

- COMMIT
- ROLLBACK
- SAVEPOINT

### Data Query Language (DQL)

- SELECT Statement
- WHERE Clause
- Comparison Operators
- Logical Operators

### Subqueries

- ANY Operator
- ALL Operator

### Aggregate Functions

- COUNT()
- SUM()
- AVG()
- MIN()
- MAX()

### GROUP BY Clause

- Grouping records
- Aggregate calculations based on groups

### SQL Built-in Functions

- ABS()
- POWER()
- EXP()
- FLOOR()
- ROUND()
- TRUNC()
- MOD()
- ASCII()

---

## Operations Performed

- Inserted new records
- Updated existing records
- Created Savepoints
- Rolled back transactions
- Retrieved records using conditions
- Performed aggregate calculations
- Grouped data using GROUP BY
- Used mathematical functions in SQL

---

## Learning Outcome

Through this session, I learned how SQL manages transactions, performs calculations on data, and retrieves information efficiently using aggregate functions and conditional queries.

Understanding TCL and aggregate functions is important for database management and technical interviews.
